# swift-watch

Watches over your Swift project's source.

---

This is early work in progress. Stay tuned!

# Installation

1. Build
2. Install in `$PATH` (such as in `/usr/local/bin/`)
3. Run `$ cd /path/to/swift/package/`
3. Run `$ swift watch -x build`
